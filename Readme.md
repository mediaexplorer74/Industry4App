# Industry4App v1.0 pre-alpha - ветка master
Псевдо-PWA-Клиент для сайта https://trends.rbc.ru/trends/industry. Сделано ИИ Qwen за 3 минуты. :) 

## О программке
UWP-приложения Industry4App - это что-то вроде ридера новостей по трендам индустрии 4.0 от RBC. Может, кого вдохновит в режиме вайб-кодинга починить "долгострои" типа моей "репки" 4PDAClient?... ;)

## Тех. моментики
Я использую Win. SDK 19041 и Min. Win. SDK 10240, чтобы обеспечить совместимость с ранней-сборкой 10240 ретро-ОС Windows 10 Mobile. 

## Тудушки
Почему-то на главной странице у меня список новостей пуст. Для парсинга применяю пакет HtmlAgilityPack (он уже подключен в проектный файл). Если что-то надо от меня (например, образец html данных, который ну никак не парсится в список articles), сообщи мне. 


## Как чинюкать
- Например, подцепить Qwen к экстеншну Kilocode в ide-шку VS Code. 

## Пример промта по исправлению моих багов 
Привет! Хочу продолжить портировать в UWP отреверсенный код wp7-приложения OpenTrackBETA (это простейший муз. клиент по свободной музыке). Касаемо UWP: Я использую Win. SDK 19041 и Min. Win. SDK 10240, чтобы обеспечить совместимость с ранней-сборкой 10240 ретро-ОС Windows 10 Mobile. Разберись, пожалуйста со всеми артефактами wp7.1/silverlight. Осовремень по мере воспроизведение музыки с задействованием MediaPlayer или MediaElement, пожалуйста. Также все устаревшие стили, содержащие слово Phone, или удали, или замени на подходящие для uwp. 

Сборку производи строго командой msbuild, а не dotnet build!
MSBuild.exe "OpenTracksBETA.sln" /p:Configuration=Debug /p:Platform=x64 /t:Build

( Если необходимо, при куче странных ошибок вроде System.String…, перед сборкой дополнительно выполни первоначальное восстановление пакетов командой

MSBuild.exe "OpenTracksBETA.sln" /t:Restore )

 Сборку производи строго командой msbuild, а не dotnet build!
MSBuild.exe "Industry4App.sln" /p:Configuration=Debug /p:Platform=x64 /t:Build
( Если необходимо, при куче странных ошибок вроде System.String…, перед сборкой дополнительно выполни первоначальное восстановление пакетов командой
MSBuild.exe "Industry4App.sln" /t:Restore )
