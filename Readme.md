## Тех. моментики
- Я использую Win. SDK 19041 и Min. Win. SDK 10240, чтобы обеспечить совместимость с ранней-сборкой 10240 ретро-ОС Windows 10 Mobile.

## Тудушки

## Пример промпта по исправлению моих багов 
Привет! Мастерю клиент для сайта https://trends.rbc.ru/trends/industry. Название моего UWP-приложения - Industry4App (это что-то вроде ридера новостей по трендам индустрии 4.0 от RBC). Касаемо UWP: Я использую Win. SDK 19041 и Min. Win. SDK 10240, чтобы обеспечить совместимость с ранней-сборкой 10240 ретро-ОС Windows 10 Mobile. Разберись, пожалуйста, почему на главной странице у меня список новостей пуст. Для парсинга применяю пакет HtmlAgilityPack (он уже подключен в проектный файл). Если что-то надо от меня (например, образец html данных, который ну никак не парсится в список articles), сообщи мне. 

Сборку производи строго командой msbuild, а не dotnet build!

MSBuild.exe "Industry4App.sln" /p:Configuration=Debug /p:Platform=x64 /t:Build

( Если необходимо, при куче странных ошибок вроде System.String…, перед сборкой дополнительно выполни первоначальное восстановление пакетов командой

MSBuild.exe "Industry4App.sln" /t:Restore )
